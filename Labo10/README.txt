## Laboratoire 10

Voici les instructions nécessaires au bon fonctionnement du laboratoire.

1) Ouvrir le dossier labo10_server et exécuter $npm install, puis $node index.js dans le terminal.
Le serveur est en cours d'exécution sur le port 3000 par défaut.

2) Si désiré, il est possible de créer un nouvel utilisateur. Pour ce faire,  aller sur Postman, faire un POST avec le
url suivant: http://localhost:3000/users. Un nouvel identifiant utilisateur vous y est retourné. Inscrire cet identifiant
à la ligne 3 du fichier api.js situé au chemin suivant: Labo10/labo10_client/src/api.js.

3) Ouvrir le dossier labo10_client et exécuter $npm install, puis $npm start dans le terminal.
L'application est en cours d'exécution sur le port 8080 par défaut.

4) Enjoy!
